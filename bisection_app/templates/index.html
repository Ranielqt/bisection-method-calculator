Mathematical Discussion

The Bisection Method is a reliable root-finding algorithm formulated upon the Intermediate Value Theorem (IVT) for continuous functions.

📖
Intermediate Value Theorem: If a function \( f(x) \) is continuous on a closed interval \( [a, b] \), and \( f(a) \) and \( f(b) \) have opposite signs (\( f(a) \cdot f(b) < 0 \)), then there exists at least one real value \( c \) in the open interval \( (a, b) \) such that \( f(c) = 0 \).

Step-by-Step Algorithm:

  1. Identify bracket boundaries \( a \) and \( b \) such that \( f(a) \cdot f(b) < 0 \).
  2. Calculate the interval midpoint: \( c = \frac{a + b}{2} \).
  3. Compute the function value \( f(c) \).
  4. Evaluate stopping criteria: if \( |b - a| / 2 < \varepsilon \) or \( f(c) \approx 0 \), then \( c \) is the root.
  5. Update the active boundary:
    • If \( f(a) \cdot f(c) < 0 \), the root resides in the left half; set \( b = c \).
    • Otherwise, the root resides in the right half; set \( a = c \).
  6. Repeat from step 2.

Convergence Rate:

The method is guaranteed to converge linearly, with the bracket error halving at every successive iteration: \( E_k = \frac{|b - a|}{2^k} \).

Advantages

Guaranteed convergence; high numerical robustness; does not require evaluating derivatives.

Disadvantages

Slow linear speed; fails to solve double root structures such as \( f(x) = (x-1)^2 \).
